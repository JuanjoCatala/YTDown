>NOTE: Url MUST be in quotes (""). Some characters could confuse the shell

`$ python3 -v -u [VIDEO URL] -o [OUTPUT PATH]`

`$ python3 -p -u [PLAYLIST URL] -o [OUTPUT PATH]`
